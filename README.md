# VIBE//SHIFT ⚡

> **Pisz mniej kodu. Twórz więcej.**

VIBE//SHIFT to nowoczesny, interaktywny landing page poświęcony filozofii **vibe codingu** i programowaniu z udziałem sztucznej inteligencji (AI-assisted coding). Zamiast skupiać się na pisaniu powtarzalnej składni, strona edukuje, jak stać się "dyrygentem" dla agentów AI takich jak Google Gemini, ChatGPT, czy Claude Code.

## 🚀 Technologie

Projekt został zbudowany z myślą o maksymalnej wydajności i płynności interfejsu (premium feel), korzystając z nowoczesnych rozwiązań:
- **HTML5 & CSS3** (Vanilla) – Nowoczesny layout (CSS Grid / Flexbox), zaawansowane maski, gradienty oraz typografia (Google Fonts).
- **JavaScript (ES6)** – Czysta logika do obsługi animacji i scrollingu.
- **GSAP & ScrollTrigger** – Do zaawansowanych, wydajnych animacji pojawiających się podczas przewijania.
- **Lenis** – Do obsługi "smooth scroll", zapewniającego idealnie płynne przewijanie strony na każdym urządzeniu.

## 🌟 Główne Funkcjonalności

- **Mroczny, cybernetyczny klimat (Dark Theme):** Neonowe gradienty i minimalistyczne komponenty graficzne imitujące konsolę.
- **Interaktywna oś czasu (The Build Loop):** Płynnie animowana ścieżka od pomysłu do wdrożenia kodu.
- **Responsive Web Design:** Strona idealnie dopasowuje się do każdego ekranu (od smartfona po monitory ultrawide).
- **Semantyczny kod:** Poprawne użycie tagów HTML5 i zachowanie wytycznych dostępności.

## 🛠️ Uruchomienie lokalne

Projekt nie wymaga żadnego skomplikowanego środowiska ani procesu budowania (build process).

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/apkmasondev/vibe_shift.git
   ```
2. Wejdź do folderu:
   ```bash
   cd vibe_shift
   ```
3. Otwórz plik `index.html` bezpośrednio w przeglądarce lub użyj rozszerzenia typu **Live Server** (w VS Code / Cursor), aby zapewnić automatyczne odświeżanie.

## 🌐 Wdrożenie (Deployment)

Strona jest w 100% statyczna. Najszybszą i darmową metodą hostowania jest **GitHub Pages**:
1. W ustawieniach swojego repozytorium przejdź do zakładki **Settings > Pages**.
2. W sekcji **Source** wybierz **Deploy from a branch**.
3. Jako gałąź (branch) wskaż `main` (lub `master`) i kliknij **Save**.
4. Twoja strona będzie dostępna online w ciągu kilku minut!
